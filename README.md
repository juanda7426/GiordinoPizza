# GiordinoPizza

**GiordinoPizza** es una landing page de alta gama diseñada para una pizzería artesanal premium. Este proyecto combina una estética visual cautivadora con una lógica de personalización avanzada para ofrecer una experiencia de usuario excepcional desde el primer clic hasta el pedido final por WhatsApp.

---

## 🚀 Características Principales

### 💎 Experiencia Visual Premium

- **Hero Dinámico:** Carrusel de imágenes en alta definición con transiciones suavizadas (Cross-fade) y branding integrado.
- **Diseño Moderno:** Uso de *Glassmorphism*, degradados de malla (Mesh Gradients) y tipografía sofisticada que evoca la tradición artesanal.
- **Animaciones Fluidas:** Implementación de `Framer Motion` para efectos de parallax y revelación de secciones al hacer scroll.

### 🍕 Sistema de Personalización Inteligente

- **Configurador de Pizzas:** Lógica avanzada que permite al usuario seleccionar el tamaño (Personal, Mediana, Familiar) y el tipo de masa (Tradicional, Delgada, Borde de Queso).
- **Gestión de Ingredientes Extra:** Interfaz interactiva para añadir adiciones (extra queso, carnes premium, vegetales) con actualización automática del costo.
- **Especialidades Mitad y Mitad:** Sistema de validación para armar pizzas con dos especialidades distintas, controlando la combinación correcta de ingredientes básicos.

### 🛒 Carrito y Checkout Optimizado

- **Edición en Tiempo Real:** Los usuarios pueden modificar los ingredientes, tamaños o cantidades de sus pizzas directamente desde el carrito sin perder el progreso.
- **Integración con WhatsApp:** Generación automática de un mensaje estructurado y detallado con el desglose del pedido (especificaciones de la pizza, dirección de envío y método de pago).
- **Interfaz Intuitiva:** Notificaciones dinámicas con `SweetAlert2` para una retroalimentación clara, limpia y elegante al agregar o eliminar productos.

---

## 🛠️ Stack Tecnológico

- **Frontend:** React.js + Vite
- **Estilos:** CSS3 Vanilla + Bootstrap 5 (React-Bootstrap)
- **Animaciones:** Framer Motion
- **Iconografía:** Lucide React
- **Alertas:** SweetAlert2
- **Gestión de Estado:** Context API de React
