# Dulce Bocado - Velvet Scoop 🍨✨

**Dulce Bocado** es una landing page de alta gama diseñada para una heladería artesanal premium. Este proyecto combina una estética visual cautivadora con una lógica de personalización avanzada para ofrecer una experiencia de usuario excepcional desde el primer clic hasta el pedido final por WhatsApp.

---

## 🚀 Características Principales

### 💎 Experiencia Visual Premium

- **Hero Dinámico:** Carrusel de imágenes con transiciones suavizadas (Cross-fade) y branding integrado.
- **Diseño Moderno:** Uso de _Glassmorphism_, degradados de malla (Mesh Gradients) y tipografía de lujo.
- **Animaciones Fluidas:** Implementación de `Framer Motion` para efectos de parallax y revelación al hacer scroll.

### 🍦 Sistema de Personalización Inteligente

- **Gestión de Bolas de Helado:** Lógica avanzada que controla la cantidad de bolas incluidas por producto y permite la adición de "Bolas Extra".
- **Distribución de Sabores:** Interfaz interactiva para que el usuario asigne cantidades específicas a cada sabor artesanal.
- **Limitación de Complementos:** Sistema de validación para salsas y toppings, asegurando la combinación perfecta.

### 🛒 Carrito y Checkout Optimizado

- **Edición en Tiempo Real:** Los usuarios pueden ajustar sus elecciones directamente desde el carrito.
- **Integración con WhatsApp:** Generación automática de mensajes detallados con el desglose del pedido, datos de envío y método de pago.
- **Interfaz Intuitiva:** Notificaciones con SweetAlert2 para una retroalimentación clara y elegante.

---

## 🛠️ Stack Tecnológico

- **Frontend:** React.js + Vite
- **Estilos:** CSS3 Vanila + Bootstrap 5 (React-Bootstrap)
- **Animaciones:** Framer Motion
- **Iconografía:** Lucide React
- **Alertas:** SweetAlert2
- **Gestión de Estado:** Context API de React

## 📦 Instalación y Uso

1. **Clonar el repositorio:**

   ```bash
   git clone [url-del-repositorio]
   ```

2. **Instalar dependencias:**

   ```bash
   npm install
   ```

3. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```
